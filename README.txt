
# Iglesia Lumbreras En La Oscuridad – Sitio web estático

Este paquete contiene un sitio listo para publicar (HTML + CSS) con:
- Página principal con misión, visión, doctrina, horarios y ubicación.
- Botón al formulario de membresía y QR integrado.
- Contacto y redes sociales.

## Cómo publicarlo (opciones)

### Opción A: Netlify (fácil y gratis)
1. Entra a https://app.netlify.com y crea cuenta.
2. Clic en **Add new site** → **Deploy manually**.
3. Arrastra la carpeta `site_lumbreras/` completa.
4. ¡Listo! Te dará una URL (puedes conectar un dominio propio).

### Opción B: GitHub Pages
1. Crea un repo en GitHub.
2. Sube todo el contenido de `site_lumbreras/`.
3. En **Settings → Pages**, selecciona **Deploy from a branch** sobre `main / root`.
4. Abre la URL que GitHub te muestre.

### Opción C: Cloudflare Pages
1. Entra a https://pages.cloudflare.com/
2. Conecta un proyecto y sube la carpeta `site_lumbreras/`.
3. Publica con 1 clic.

### Editar enlaces de redes
En `index.html`, busca la sección **Redes** y cambia los `href="#"` por tus enlaces reales.

### Cambiar horarios, doctrina o texto
Edita `index.html` y guarda. Vuelve a publicar en tu plataforma.
